# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the
work. Harassment and discrimination are not tolerated.

Report unacceptable behavior via GitHub (a private report to the maintainers).
Maintainers may remove, edit, or reject contributions that violate this code.

The full text is available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
